# Credit Card Fraud Detection using Machine Learning

## Overview
This project builds a machine learning model to detect fraudulent credit card transactions using Logistic Regression.

The dataset contains 284,807 transactions with 492 fraud cases, making it a highly imbalanced dataset.

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Google Colab

## Project Workflow
1. Data loading
2. Data exploration
3. Handling imbalanced data using undersampling
4. Feature and target separation
5. Train-test split
6. Model training using Logistic Regression
7. Model evaluation

## Model Performance
Training Accuracy: **94.15%**

Testing Accuracy: **92.89%**

## Dataset
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
